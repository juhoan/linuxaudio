# Documentation

The script launches Spotify and REAPER and configures wiring so that outputs from both softwares are connected to inputs of `Scarlett Solo (3rd Gen.) Direct Scarlett Solo USB` device. Now you can listen to Spotify and record/play on REAPER at the same time.

Software required:
* Spotify desktop app
* REAPER Digital Audio Workstation (DAW)
* Pipewire
* qpwgraph
