# Documentation

The script launches Spotify and REAPER and configures wiring so that outputs from both applications are connected to inputs of `Scarlett Solo (3rd Gen.) Direct Scarlett Solo USB` device. Now you can listen to Spotify and record/play on REAPER at the same time.

## Software required

* Spotify desktop app
* REAPER Digital Audio Workstation (DAW)
* Pipewire
* qpwgraph

## Hardware required

* PC (I'm using Lenovo T15p laptop)
* Scarlett Solo (3rd Gen) USB sound card
* Guitar (I'm using PRS S2 Custom 35th anniversary edition)
* Instrument cable to connect the guitar to the sound card
* Headphones (I'm using Bose QuietComfort 45)
* Headphone cable(s) to connect the USB sound card to headphones
  * USB sound card output is 6.3 mm
* USB cable for connecting sound card to laptop
  * Sound card output is USB-C
  * Computer input is USB-A

## Running
```
./spotify-reaper.sh
```
